# -
مشروع نظام مدرسة
  نظام إدارة الطلاب

  الوصف

برنامج بايثون لإدارة معلومات الطلاب والدروس باستخدام قاعدة بيانات SQLite. يمكن للمستخدم إضافة، حذف، تعديل، وعرض معلومات الطلاب.

  المتطلبات

* Python 3.x
* مكتبة sqlite3 (موجودة ضمن بايثون بشكل افتراضي)

  طريقة التشغيل

1. افتحي الطرفية (Terminal) أو موجه الأوامر (Command Prompt)
2. شغلي البرنامج:

```
python main.py
```

  الوظائف

* إضافة طالب وربطه بالدروس
* حذف طالب
* تعديل معلومات الطالب
* عرض معلومات الطالب بما في ذلك الدروس

  ملاحظات

* يمكن لكل طالب التسجيل في أكثر من درس
* يمكن لكل درس أن يشمل أكثر من طالب
* جداول قاعدة البيانات: students, lessons, student_lessons
